# FolderTools
Módulo do Power Shell para visualização detalhada de pastas e arquivos no Windows bem como cálculo do total.

Rode o Script "install.ps1 para Baixar e implatar o pacote

No diretorio onde vc baixar o script,
Rode No terminal Powershell como administrador: 

Set-ExecutionPolicy -Scope LocalMachine -ExecutionPolicy RemoteSigned -Force; .\install.ps1
Se vc fizer certo, o Terminal irá imprimir essa mensagem:

PS C:\Users\SEU_USUARIO\Documents> Set-ExecutionPolicy -Scope LocalMachine -ExecutionPolicy RemoteSigned -Force; .\install.ps1
Baixando modulo FolderTools do GitHub...
Download conclui­do.
Modulo instalado em: C:\Users\SEU_USUARIO\Documents\PowerShell\Modules\FolderTools
FolderTools carregado com sucesso!
